# Prodesk IT - Digital Agency Landing Page

This project is a landing page for a fictional digital marketing agency called "Prodesk IT". It was created as part of an internship assignment to demonstrate proficiency in HTML, CSS, and responsive design.

## Live URL (Deployed using github pages)

[Link to live site](https://bugsfounder.github.io/ProdeskIT/)

## Difficulty Level Completed: Level 2

This project fulfills all the requirements for Level 2. (With Glassmorphism effect mentioned in Level 3)

## Features

- **Responsive Design:** The layout is fully responsive and looks great on both mobile and desktop devices.
- **Sticky Navbar:** The navigation bar remains fixed at the top of the page for easy access.
- **Hero Section:** A prominent hero section with a catchy headline and a call-to-action button.
- **Services Section:** A clean and modern grid displaying the agency's services.
- **Dark Mode:** A toggle in the navbar allows users to switch between light and dark themes.
- **Micro-Interactions:**
  - Buttons have hover effects.
  - Service cards lift on hover.
- **Footer:** A simple footer with copyright information and social media links.

## Technologies Used

- HTML
- CSS
- JavaScript
